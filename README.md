<h1>iConverse iOS App</h1>

<p>This is an app that calculates the conversions of some unit types</p>

<h2> Unit Types to be converted are:</h2>
<ol>
  <li>Length</li>
  <li>Volume</li>
  <li>Temperature</li>
  <li>Time</li>
</ol>
